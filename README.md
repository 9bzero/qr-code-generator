# qr-code-generator

Generate QR codes for URLs, plain text, or vCard contacts.

## Features

- URL mode: paste a link, get a QR code
- Text mode: encode any plain text
- vCard mode: name, phone, email, URL → scannable contact card
- Adjustable size and error correction level
- Download as PNG or SVG

## Run

```bash
npm install && npm run dev
```
